# Matrix Assessment – Value Generation Page

A full-stack web application built using **React.js** for the frontend and **Node.js/Express.js** for the backend.

## Features

### Frontend
- Hero Banner
- Services Section
- Value Generation Framework
- Client Testimonials
- Contact Form
- Responsive Design

### Backend
- Contact Form API
- Input Validation
- Form Submission Storage
- Basic Rate Limiting for Spam Prevention

---

## Prerequisites

### Required Node.js Version
- Node.js v18.x or above
- npm v9.x or above

Check your version:

```bash
node -v
npm -v
```

---

## Environment Variables

> No additional environment variables are required.

## Run Frontend

```bash
cd frontend
npm install
npm run dev
```

Frontend runs at:

```text
http://localhost:5173
```

---

## Run Backend

```bash
cd backend
npm install
npm run start
```

Backend runs at:

```text
http://localhost:5000
```

The Contact Form API supports:

- Submission storage
- Basic rate limiting to prevent spam
